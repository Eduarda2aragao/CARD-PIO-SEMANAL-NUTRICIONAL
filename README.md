🍎 Cardápio Semanal Nutricional com Python

Projeto acadêmico desenvolvido em Python utilizando técnicas de Inteligência Artificial para gerar cardápios semanais automatizados com foco em emagrecimento e ganho de massa muscular.

📌 Objetivo

O sistema foi criado para auxiliar usuários na organização de dietas equilibradas através da geração automática de planos alimentares.

A aplicação utiliza conceitos de:

Busca em Largura (BFS)
Algoritmos Genéticos
Heurísticas
Espaço de Estados
🧠 Tecnologias Utilizadas
Python 3
Programação Orientada a Objetos
Inteligência Artificial
Algoritmos de Busca
⚙️ Funcionalidades

✅ Gerar cardápios semanais

✅ Separação por refeições:

Café da manhã
Lanche
Almoço
Jantar

✅ Planejamento para:

Emagrecimento
Ganho de massa

✅ Avaliação nutricional baseada em:

Calorias
Proteínas
Variedade alimentar

🧬 Algoritmo Genético

O Algoritmo Genético foi utilizado no cenário de emagrecimento.

Cada cromossomo representa uma dieta semanal.

Parâmetros utilizados
População inicial aleatória
Taxa de crossover
Taxa de mutação
Critério de parada
Objetivo

Gerar soluções mais variadas e eficientes para o usuário.

📚 Busca em Largura (BFS)

A BFS foi aplicada no cenário de ganho de massa.

Vantagens

📊 Heurística

A função heurística avalia:

Diferença entre calorias consumidas e meta

Quanto menor o erro, melhor a solução.
